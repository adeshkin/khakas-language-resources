### Parallel corpus:
- located - **khakas_texts_with_russian_translation/data**
- _kjh_ - khakas
- _ru_ - russian
- source - https://khakas.altaica.ru/texts

```python
import pandas as pd

mode = 'train'
path = f'khakas_texts_with_russian_translation/data/{mode}.csv'
df = pd.read_csv(path)
pairs = df[['kjh', 'ru']].values.tolist()
print(pairs[0])
```
